📊 Bank Loan Analysis Dashboard

📌 Project Overview
This project analyzes bank loan data to evaluate loan applications, funding distribution, repayment performance, and default risk. The dashboard provides interactive insights to support data-driven decision-making in the banking sector.

🛠️ Tools & Technologies
Power BI (Dashboard Development & Data Modeling & Data Cleaning)
SQL (Data Extraction & Querying)
